# CSC8503

## Controls

WASD - Move
Mouse - Look

1-4 Attach pane corner to where you are looking

SHIFT + 1-4 Reel in the corner
CTRL + 1-4 Let out the corner
SHIFT + CTRL + 1-4 Detach the corner

## Networking

The dedicated server is not working, however a client can host a game and other clients can connect to them.

## Levels

Level one has a basic scene with some physics objects to interact with.

Level two has a basic maze to navigate, with two AI enemies.
